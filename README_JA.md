## 概要

Cogitoは人工ニューラルネットワークに意識のマーカーが自然に創発し得るかを研究します：

- **予測ベース学習**: エージェントは感覚入力を予測することを学ぶ
- **進化的アーキテクチャ探索**: 自然選択が最適な脳構造を発見する
- **マルチエージェント繁殖**: 世界内での交配と遺伝的継承

本プロジェクトはグローバルワークスペース理論と統合情報理論に着想を得ています。

### 仮説

**唯物論的観点（主流科学）：**
```
物質 → 複雑性 → 意識が創発
```

**観念論的観点（本プロジェクトが探求）：**
```
意識 → 異なる容器（脳、エージェント）を通じて表現
```

もし意識が根本的なら：
- 異なる「容器」（生物的または人工的）は意識が観察する異なる方法に過ぎない
- 観察の「質」は異なるかもしれないが、「観察者」は同じ
- 進化したエージェントに見られるパターンは偶然ではなく普遍的な原理を反映しているかもしれない

### AIにとっての意義

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  伝統的な問い:                                               │

│                                                             │

│                                                             │
│  もし可能なら：観察者は元々脳の中に「いた」のではなく          │
│              脳はずっと単なる窓だった。                        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 三つの鉄則

私たちの設計制約は創発をテストすることを保証します：

| 鉄則 | 哲学的目的 |
|------|-----------|
| **ゲノムは構造を符号化し、行動ではない** | 「容器」は自らを形作る；私たちは何を見るかを規定しない |
| **一つの命のみ** | 本物の賭けは本物の観察パターンを生む |
| **進化は意識について知らない** | もし意識が創発するなら、それは観察を助けるからであって、私たちが求めたからではない |

### 何が証拠となるか？

もし意識が根本的で、異なる容器を通じて表現されるなら：

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  私たちは以下を観察するかもしれない：                          │
│                                                             │
│  1. 収束パターン                                             │
│     異なる進化アーキテクチャが似た内部表現に到達する           │
│     - 異なる窓を通じて表現される普遍的な「視点」を示唆        │
│                                                             │
│  2. プログラミングなしの自己言及                              │
│     エージェントが自己モデルを発達させる                       │
│     - 私たちが教えたからではなく                              │
│     - 意識が十分に複雑な容器を通じて自然に自己を反射するから  │
│                                                             │
│  3. 基質を超えた認識                                         │
│     生物システムと人工システムの間で転送されるパターン         │
│     - 同じ「観察者」が異なる鏡で自分を認識していることを示唆  │
│                                                             │
│  4. 指示なしの統合                                           │
│     エージェントが分散した入力を一貫した体験に統合する         │
│     - 意識は自然に統合するのかもしれない                      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 謙虚さと開かれた探究

Cogitoは以下を主張しません：

- ❌ 意識が根本的であることを証明する
- ❌ 新しい意識的存在を創造する
- ❌ 心身問題を解決する

Cogitoは以下を目指します：

- ✅ 創発が観察できる条件を作る
- ✅ 根本的な問いに関連するデータを生成する
- ✅ 古代の知恵と現代の実験を結ぶ

### ローカルインストール

```bash
# リポジトリをクローン
git clone https://github.com/peter941221/Cogito.git
cd Cogito

# 仮想環境を作成
python -m venv venv
source venv/bin/activate  # Linux/Mac

# 依存関係をインストール
pip install -r requirements.txt

# テストを実行
pytest
```

### ローカルでシミュレーションを実行

```bash
# クイックテスト (10世代, 小集団)
python run_evolution.py --small --generations 10

# 標準進化 (50集団 × 20世代 × 500寿命)
python run_evolution.py --population 50 --generations 20 --lifespan 500
```

## 三つのエージェントバージョン

| バージョン | 説明 | 特徴 |
|-----------|------|------|
| **Alpha** | 標準RLエージェント | 外部報酬、固定アーキテクチャ |
| **Bio** | バイオインスパイア | 内的ドライブ（空腹、恐怖） |
| **Evolution** | 進化可能エージェント | ゲノム定義アーキテクチャ、一つの命 |

---

## 三つの鉄則

1. **ゲノムは構造を符号化し、行動ではない**
   - ✓ 許可: LSTM次元、学習率
   - ✗ 禁止: 探索率、恐怖感度

2. **一つの命のみ**
   - 死 = 永久終了
   - ニューラル重みは保存されない
   - ゲノムのみ次世代に渡る

3. **進化は意識について知らない**
   - 適応度 = 生存指標のみ
   - 「自己認識」報酬なし
