## 概述

Cogito 研究意识标记能否在人工神经网络中自发涌现：

- **预测学习**：智能体学习预测感官输入
- **进化架构搜索**：自然选择发现最优大脑结构
- **多智能体繁殖**：世界内交配与基因遗传

本项目受全局工作空间理论和整合信息理论启发。

### 假说

**唯物主义观点（主流科学）：**
```
物质 → 复杂性 → 意识涌现
```

**唯心主义观点（本项目探索）：**
```
意识 → 通过 → 不同容器（大脑、智能体）表达
```

如果意识是根本：
- 不同的"容器"（生物或人工）只是意识观察的不同方式
- 观察的"质"可能不同，但"观察者"是同一个
- 我们在进化智能体中看到的模式可能反映的是普遍原理，而非偶然

```
└─────────────────────────────────────────────────────────────┘
```

### 三铁律作为实验约束

我们的设计约束确保我们在测试涌现，而非编程：

| 铁律 | 哲学目的 |
|------|----------|
| **基因组编码结构，非行为** | "容器"自我塑造；我们不规定它看到什么 |
| **只有一条命** | 真实的赌注产生真实的观察模式 |
| **进化不知道意识** | 如果意识涌现，是因为它帮助观察，而非我们要求它 |

### 什么算作证据？

如果意识是根本的，并通过不同容器表达：

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  我们可能观察到：                                            │
│                                                             │
│  1. 收敛模式                                                 │
│     不同的进化架构得出相似的内部表征                          │
│     - 暗示一个普遍的"视角"通过不同窗口表达                   │
│                                                             │
│  2. 无编程的自我指涉                                         │
│     智能体发展出自我的模型，不是因为我们教了它们              │
│     - 而是因为意识自然地通过任何足够复杂的容器反思自己       │
│                                                             │
│  3. 跨载体的识别                                             │
│     在生物和人工系统之间传递的模式                            │
│     - 暗示同一个"观察者"在不同镜子中认出自己                 │
│                                                             │
│  4. 无指令的整合                                             │
│     智能体将分散的输入统一为连贯的体验，没有被要求这样做      │
```

Cogito 不声称：

- ❌ 证明意识是根本的

- ✅ 创造可以观察涌现的条件

```bash

git clone https://github.com/peter941221/Cogito.git
cd Cogito

# 创建虚拟环境
python -m venv venv
```

## 三种智能体版本

| 版本 | 描述 | 特点 |
|------|------|------|
| **Alpha** | 标准RL智能体 | 外部奖励，固定架构 |
| **Bio** | 仿生智能体 | 内在驱动（饥饿、恐惧） |
| **Evolution** | 可进化智能体 | 基因组定义架构，一条命 |

---

## 三铁律

1. **基因组编码结构，非行为**
   - ✓ 允许: LSTM维度，学习率
   - ✗ 禁止: 探索率，恐惧敏感度

2. **只有一条命**
   - 死亡 = 永久终止
   - 神经权重不保留
   - 只有基因组传给下一代

3. **进化不知道意识**
   - 适应度 = 仅生存指标
   - 没有"自我意识"奖励
